# Personal Portfolio Resume Website

A simple and responsive personal portfolio website built using **HTML**. This project showcases personal information, educational background, work experience, technical skills, achievements, and projects in a structured resume format.

## 📌 Features

* Personal profile section
* Contact information with clickable links
* Educational qualifications
* Work experience timeline
* Technical skills showcase
* Achievements section
* Project portfolio section
* Profile image integration
* Simple and clean HTML structure
